## 🚀 Overview

terminal-setup helps you set up a modern terminal with one script. It installs and configures tools that make the command line easier to use and easier to read.

It includes:

- Ghostty for the terminal window
- Fish for a friendly shell
- Starship for a clean prompt
- a Nerd Font for icons
- useful CLI tools for daily work

This project is made for people who want a better terminal without spending time on manual setup.
